⚡PROJETO - CÁLCULO CONSUMO ESTIMADO DE ENERGIA

OBJETIVO: CALCULAR O CONSUMO ESTIMADO DE ENERGIA DE QUALQUER APARELHO DE SUA PREFERÊNCIA.

LINGUAGEM UTILIZADA: PHYTON;

INSTRUÇÕES PARA EXECUTAR O PROGRAMA:

DIGITE O NOME DO APARELHO;

EM SEGUIDA, A POTÊNCIA;

EM SEGUIDA, O TEMPO MÉDIO DE USO DIÁRIO;

APÓS ISSO, O PROGRAMA IRÁ CALCULAR PARA VOCÊ O CONSUMO MENSAL.

FÓRMULA UTILIZADA PARA O CÁLCULO: consumoMensal = (potencia * tempo_medio * 30) / 1000
