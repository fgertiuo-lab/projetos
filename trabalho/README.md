# 📊 Pesquisa de Opinião em Python

## 📌 Sobre o projeto

Este projeto consiste em um programa simples desenvolvido em Python que realiza uma pesquisa de opinião com usuários.

O sistema coleta informações básicas como nome e idade, além de registrar a opinião do entrevistado sobre um determinado sistema, classificando-a como:

* Excelente
* Bom
* Ruim

---

## ⚙️ Funcionamento

O programa executa um laço de repetição para entrevistar várias pessoas. Durante cada interação, o usuário deve informar:

* Nome
* Idade
* Opinião (EXCELENTE, BOM ou RUIM)

Ao final da execução, o sistema apresenta a quantidade total de respostas para cada categoria.

---

## 🧠 Objetivo

O objetivo do projeto é praticar conceitos fundamentais da programação em Python, como:

* Estruturas de repetição (`for`)
* Estruturas condicionais (`if`, `else`)
* Entrada de dados com `input`
* Manipulação de variáveis e contadores
